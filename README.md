# FUTURE_CS_01 - Vulnerability Assessment on DVWA

## Overview
This repository contains the first task completed during my Future Interns Cyber Security Internship.

The project focuses on performing a vulnerability assessment on the Damn Vulnerable Web Application (DVWA) using Kali Linux and basic penetration testing tools.

## Tools Used
- Kali Linux
- Nmap
- Nikto
- Firefox Browser Developer Tools

## Key Findings
- Open MySQL Service (Port 3306)
- Missing X-Frame-Options Header
- Missing X-Content-Type-Options Header
- Server Information Disclosure
- HTTP Request and Response Analysis
- Session Cookie Analysis

## Files Included
- Vulnerability Assessment Report (PDF)
- Nmap Scan Result
- Nikto Scan Result
- HTTP Request Screenshot
- HTTP Response Screenshot
- Headers Analysis
- Cookies Analysis
