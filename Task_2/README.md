# Future Interns - Cyber Security Internship

## Task 2: Phishing Email Analysis

### Objective
Analyze email samples to identify phishing indicators and classify them as Safe, Suspicious, or Phishing.

### Project Description
This project demonstrates phishing email analysis by examining sender addresses, domains, embedded links, attachments, and social engineering techniques. Based on these observations, each email is classified and appropriate security recommendations are provided.
